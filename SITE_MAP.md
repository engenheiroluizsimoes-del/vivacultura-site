# SITE_MAP.md

Mapa permanente do site institucional do Instituto Viva Cultura.

O site atual e uma pagina unica institucional com navegacao por ancoras.

## Estrutura futura recomendada para projetos reais

Quando o Instituto tiver projetos reais publicados, recomenda-se evoluir a secao de projetos para uma estrutura com:

- nome do projeto;
- resumo curto;
- objetivo;
- publico atendido;
- local;
- periodo;
- status;
- parceiros;
- imagens proprias;
- resultados ou impacto;
- chamada para contato ou parceria.

Evitar criar banco de dados ou painel administrativo sem aprovacao explicita de escopo.

## Estrutura futura recomendada para galeria institucional

Quando houver acervo proprio, recomenda-se criar uma galeria com:

- imagens otimizadas;
- textos alternativos descritivos;
- creditos quando aplicavel;
- organizacao por projeto ou evento;
- cuidado com direitos autorais e autorizacao de uso de imagem;
- carregamento leve para mobile.

## Observacoes para manutencao

- Preservar portugues brasileiro.
- Preservar dados oficiais definidos em `PROJECT_SPEC.md`.
- Manter o site leve e responsivo.
- Respeitar `prefers-reduced-motion`.
- Validar `npm run build` antes de concluir tarefas tecnicas.
- Nao alterar dominio, DNS ou Cloudflare sem tarefa explicita.
- Nao versionar `node_modules/`, `dist/`, `.env`, tokens, senhas, credenciais ou ZIPs.
